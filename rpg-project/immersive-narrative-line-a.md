# Immersive Narrative Experience - Line A (Ch1-Ch3 Core)

**Purpose**: Showcase the "fleshed-out" emotional journey from Village Entrance to Black-Cloaked Battle
**Scope**: Ch1 (入门) → Ch2 (试炼) → Ch3 (古碑冲突) - 3 chapters, ~45 minutes playtime
**Format**: Scene-by-scene with emotional beats, sound effects, variable triggers, and branching dialogue
**Status**: Ready for Tech Implementation

---

## 🎭 沉浸线路总览：Line A "血脉觉醒之路"

**目标体验**：玩家从懵懂新人 → 经历试炼 → 发现家谱 → 遭遇黑衣人 → 谢临川牺牲，情感逐步加深（好奇→紧张→震撼→绝望→催泪）

**情绪曲线**：
```
好奇（Ch1）→ 紧张（Ch2）→ 震撼（Ch3 家谱）→ 绝望（Ch3 战斗）→ 催泪（Ch3 牺牲）
```

**总时长**：约 45 分钟（符合 5 分钟 Demo 可玩性 + 叙事密度）

## 📖 沉浸段落 1：Ch1 宗门日常 - "新人与旧影"

**场景序列**：M001（宗门入口）→ M002（谢临川初遇）→ M003（宗门广场）→ M004（师尊殿任务）→ M005（噩梦预兆）

### 体验流程

```
【M001-1】宗门入口（清晨雾霭）
- 音效: wind_rustling（风声）+ distant_birds（鸟鸣）
- 变量: GS_NEW_GAME_START = true
- 情绪: 平静但略带迷茫（玩家刚刚开始）
- 玄微子："沈照璃，从今日起你便是宗门弟子。你那枚玉佩，别离身。"
- 玩家内心: "玉佩戴了十几年，从来不知道它有什么特别..."

【M002-1】谢临川初遇（宗门广场）
- 音效: footsteps_approaching（脚步声）
- 变量: XIE_FIRST_MEET = true
- 情绪: 第一个"值得信任"的人出现
- 谢临川："新人吧？跟我来。练剑场那边，没人会打扰。"
- 潜台词: 他知道你在被排挤（玩家感到被理解）
- 玩家选项:
  A. 跟他走（ trusting）
  B. 我先逛逛（ cautious）
  → 无论选哪个，他都微笑："随你。但记住，这里不全是善意。"

【M003-2】宗门氛围（同门对话）
- 音效: chatter_background（低声议论）
- 变量: GS_CLAN_HOSTILITY = true
- 情绪: 感到被孤立
- 同门A（路过）:"沈家遗孤？别靠太近。"
- 同门B（远处）:"新人，宗门只看实力。"
- 玩家内心: "为什么他们都知道沈家的事..."

【M004-3】师尊殿任务
- 音效: temple_bells（钟声）
- 变量: GS_MASTER_QUEST_GIVEN = true
- 情绪: 使命感，但也感到不祥
- 玄微子："你根骨奇佳，但命途多舛...好自为之。你的命，不止属于你自己。"
- 玩家内心: "命途多舛？什么意思..."

【M005-6】夜晚噩梦（粉色光影）
- 音效: whispering_low（低语）+ pink_swoosh（粉色光影划过）
- 变量: GS_RESIDUE_FIRST_VISION = true
- 情绪: 恐惧+困惑
- 画面: 粉色光影中，女子侧脸（绮罗音 silhouette），轻声说："等你回来..."
- 玩家惊醒，发现玉佩在发烫
- 玩家内心: "刚才那是谁？'回来'？我从未来过这里啊..."
```

### 支线对话（可选触发）

**如果玩家在 M003 主动与谢临川再次对话**：
```
谢临川："怎么，那群人又说了什么？"
玩家选项:
  A. 没什么。（avoid）
  B. 他们提到沈家。（confront）
  C. 你为什么帮我？（direct）

→ 选 B: 谢临川眼神一闪："沈家的事...别提。但如果你想知道，三个月后宗门试炼，别死。"

```

## 📖 沉浸段落 3：Ch3 家谱调查 - "诡异与身份"

**场景序列**：M010（废弃村庄入口）→ M011（古碑外围）→ M012（石碑谜题）→ M013（密室入口）→ M014-M017（家谱调查）

### 体验流程

```
【M010-1】废弃村庄入口
- 音效: wind_howling（呼啸风）+ eerie_silence（诡异的静）
- 变量: GS_VILLAGE_ENTERED = true
- 情绪: 不安+好奇心
- 提示: "废弃村庄 - 沈家旧地，慎入"
- 怀中的玉佩开始发烫，表面浮现微光，指向古碑方向。
- 谢临川："这里不对劲...太安静了。"

【M011-2】古碑外围
- 音效: low_drone（低频嗡鸣）+ distant_whisper（远处低语）
- 变量: GS_STELE_APPROACHED = true
- 情绪: 悬疑感拉满
- 调查碑文:
  玩家: "九劫归一...是什么意思？"
  谢临川（皱眉）:"这阵法...我在古籍上见过。但具体..."
  （他话没说完，远处传来树枝断裂声）

【M012-3】石碑谜题（解开封印）
- 音效: stone_grinding（石头摩擦）+ seal_break（封印破裂）
- 变量: GS_STELE_PUZZLE_SOLVED = true
- 情绪: 紧张+成就感
- 谜题: 根据玉佩光芒调整 3 个符文（五行）
- 成功后: 石碑移开，露出地下通道

【M013-4】密室入口
- 音效: damp_dripping（潮湿滴水）+ heartbeat（心跳声）
- 变量: GS_CHAMBER_ENTERED = true
- 情绪: 进入未知的胆怯
- 谢临川："等等...这下面有极强的能量反应。你要下去吗？"
- 玩家选项:
  A. 当然，家谱可能在下面。（bold）
  B. 我们先准备一下。（cautious）
  → 无论选什么，谢临川都叹:"我就知道你会选A。走吧，我掩护你。"

【M014-M017】家谱调查（核心悬疑段落）
- 音效: suspenseful_pulse（悬疑脉冲）+ faint_glow（微光渲染）
- 变量: GS_FAMILY_REGISTRY_OBTAINED = true, GS_RESIDUE_IDENTITY_HINT = true, GS_SHADOW_SEEN = true
- 情绪: 震撼+自我认知危机

**调查流程（3 页）**:

> 第1页:
> 石室的寒气渗入骨髓。你拿起石台上的家谱，皮革封面早已失去弹性，像某种濒死生物的皮肤。翻到首页，"沈氏血脉"四字在昏暗光线下泛着诡异的暗金色。
> 
> 手指划过历代族人名字，突然停在最新一页——"沈照璃"三个字墨迹未干，仿佛昨天晚上才写上去。
> 
> 心跳漏了一拍。

> 第2页:
> 你猛翻到中间某页，纸面破裂，断句像是被人从完整页上 hastily 撕下又拼回：
> 
> "...九劫归一...血脉封印...六魂归位..."
> "...深渊之中，残我沉眠..."
> "...集齐碎片，方可重开仙路..."
> 
> 每一行字都在发烫。你忽然意识到：这些句子**不是描述残我**——它们**是在描述你**。
> 
> 边缘小字列出六人：绮罗音、绯铃、宁素心、白清漪、姬无瑕、原初照璃。

> 第3页:
> 就在你读完最后一人时，家谱背面的荧光**突然转向**，红光不偏不倚射向你胸口——心脏的位置。墙壁上的壁画应声亮起：六道身影围绕着九劫归一阵跪拜，阵心悬浮的晶体，跟你昨天在秘境边缘看到的玄牝天香体光芒**一模一样**。
> 
> 壁画下方，一行新刻的字浮现：
> "**他们都在等你回来**。"
> 
> "回来"？
> 
> 你从未去过这里，何来"回来"？
> 
> 身后传来脚步声。你猛地转身，谢临川站在门口，脸色在阴影中看不清。但让你后背发凉的是——他的影子，在墙上**多出了一道轮廓**，像是有人站在他身后，又迅速消散。
> 
> 谢临川问："怎么了？"
> 
> 你想说家谱上的事，但喉咙像被那个"回"字堵住了。**有些秘密，说出来就会变成真的**。而你现在还不敢确认，哪个"你"才是真的。
> 
> 家谱在你手中微微震颤，仿佛有生命。

- 变量设置在每页结束时触发
- 玩家调查后自动进入思考状态（5秒无操作）
```

### 支线对话

**如果玩家在 M011（古碑外围）尝试离开路线**:
```
谢临川（拦路）:"你要去哪？"
玩家:
  A. "我觉得不安全，想回去。" → 谢临川:"那你可以一个人回。但家谱...真的不重要吗？"
  B. "没什么，继续。"（继续主线）
→ 选 A 设置 GS_PLAYER_HESITATION = true，影响 later 对话
```

## 📖 沉浸段落 4：Ch3 黑衣人冲突 - "压迫与反转"

**场景序列**：M018（返回村庄）→ M019（黑衣伏击）→ M020-M023（战斗）→ M024（败北）→ M025（谢临川独白）

### 体验流程

```
【M018-1】返回村庄（紧张氛围）
- 音效: ominous_strings（不祥弦乐）+ footstep_crunch（脚步踩碎枯枝）
- 变量: GS_RETURN_JOURNEY = true
- 情绪: 知道有危险，但必须回
- 提示: "空气中的杀气越来越浓...谢临川走在前面，步伐放慢。"
- 谢临川（低声）:"跟紧我。如果出事，别回头。"

【M019-2】黑衣伏击（阴影现身）
- 音效: sudden_silence（突然寂静）+ cloak_swish（斗篷摆动）
- 变量: GS_BLACKCLOAK_ENCOUNTERED = true
- 情绪: 窒息般的压迫
- 黑衣人从树后现身，全身包裹在黑色斗篷中，只露出一双冰冷的眼睛。
- 黑衣人:"沈家血脉...竟还活着。把东西交出来。"
- 玩家内心: "他直奔主题...家谱在我身上！"

【M020】黑衣人突袭（战前对话）
- 音效: battle_theme（战斗主题）+ low_choir（低沉合唱）
- 变量: GS_BLACKCLOAK_DIALOGUE = true
- 情绪: 恐惧但必须战斗
- 对话:
  沈照璃:"你是什么人？为什么要灭我沈家？"
  黑衣人:"答案，下面自然知道。现在，要么交出血脉信物，要么——永绝后患。"
  谢临川（挡在前）:"有我在，别想动她！"
  黑衣人:"不自量力。\n™ SEL 的后人，你也配在这种地方？"
- 玩家注意到:"™ SEL？那是什么..."
- 变量: GS_TEAM_SEL_HINT = true（谢临川身份伏笔）

【M021-023】必败战斗
- 音效: intense_combat（激烈战斗）+ desperate_hits（绝望攻击）
- 变量: GS_BATTLE_DEFEAT_IMPENDING = true
- 情绪: 无力感+绝望
- 战斗设计:
  - 玩家攻击黑衣人: 强制伤害 1 点（防御极高）
  - 黑衣人攻击: 玩家 HP 每回合 -30%
  - 第三回合: 黑衣人使用大招，"Shadow Bind"（玩家无法行动 1 回合）
- 玩家尝试: 
  - 逃跑（失败，提示"无法逃离"）
  - 防御（效果微弱）
  - 道具（全部无效）
- 第 4 回合: 玩家 HP ≤20%，黑衣人最后一击
- 音效: critical_hit（致命一击）+ slow_motion（慢动作）
- 黑衣人:"太弱了。就凭你，也敢妄想继承沈家？"

【M024】败北触发
- 音效: defeat_choir（失败合唱）+ body_hit（身体倒击声）
- 变量: GS_DEFEATED_BY_BLACKCLOAK = true
- 情绪: 绝望+无力
- 画面: 主角被击飞，撞在石碑上，吐血
- 黑衣人走向主角，伸手要家谱
- 谢临川突然冲出，挡在主角身前
- 谢临川:"想动她，先过我。"
- 黑衣人冷笑:"™ SEL 的后人，你竟敢擅自介入沈家的命定轨迹。"
- 谢临川沉默，但没有让开

【M025】谢临川独白（3 页催泪）
- 音效: sad_piano（悲伤钢琴）+ rain_starting（雨声渐起）
- 变量: GS_XIE_SACRIFICE_UNDERSTOOD = true, GS_RESIDUE_MECHANISM_CLARIFIED = true
- 情绪: 感动+悲伤+理解
- 第1页:
  谢临川（靠墙，咳出一口血）:
  "我本来...能带你安全离开的。"
  你握紧拳头。这句话的意思是——他**没**带你走。
- 第2页:
  谢临川:
  "但如果你没亲眼见到黑衣人的实力...就不会明白沈家的敌人有多强。\n你也不会...真正理解残我是什么。"
  你愣住。他**知道**残我？
- 第3页:
  谢临川（闭眼，声音微弱）:
  "我父亲临终前告诉我——'沈家的女儿必须自己走过古碑。必须亲眼见到第一个残我。必须...失去一些东西。'\n我以为...那些只是传说。"
  他睁开眼，第一次你看到他眼中有**泪光**:
  "我以为我能保护你不失去任何东西。但原来...我必须成为那个'失去'。"
  主角内心:
  "他在说...他的受伤是**计划的一部分**？为了保护我，他让自己变成我必须付出的代价？"
- 结束后: 自动等待 10 秒（玩家消化）
- 变量: GS_XIE_SACRIFICE_RECOGNIZED = true
```

### 支线对话

**如果玩家在 M020（黑衣人对话）选择质问他**:

## 📖 沉浸段落 5：Ch3 尾声 - "败逃与追问"

```
【M026-1】逃离村庄（夜）

- 变量: GS_ESCAPE_SUCCESS = true

- 提示: "你和谢临川一路狂奔，直到村庄的灯火彻底消失..."
- 谢临川（受伤但仍微笑）:"抱歉...我答应过要保护你的。"
- 玩家内心: "别说这个...你先保住自己。"

【M027-2】安全地带（山洞）
- 音效: cave_drip（滴水声）+ crackling_fire（篝火）
- 变量: GS_SAFE_SPOUSE = true
- 情绪: 疲惫+思考
- 给谢临川包扎，处理伤口
- 谢临川:"家谱...在你身上吗？"
- 玩家检查背包: "在。但它越来越烫..."
- 谢临川（目光锐利）:"它在回应你。你体内的残我，也开始苏醒了。"

【M028-3】家谱复盘（信息整合）
- 音效: page_turn（翻页声）+ mysterious_hum（神秘嗡嗡）
- 变量: GS_REGISTRY_REANALYZED = true
- 情绪: 恍然大悟
- 玩家重新阅读家谱，结合谢临川的话:
  - "六魂归位" → 六残我
  - "集齐碎片" → 需要收集
  - "深渊之中" → 下一站
```

## 🎯 设计意图说明

1. **每段落都有明确情绪目标**：不是随意推进，而是刻意制造"好奇→紧张→震撼→绝望→催泪→决心"的递进
2. **变量同步**：每个段落触发多个变量，确保：
   - Playtest 可测量（TP-01~TP-05 对应的变量都在此线中触发）
   - 后续章节可读取历史状态（如 RESIDUE_QUESTION_EARLY 影响 Ch4 素问对话）
3. **音效与文字联动**：
   - 悬疑时: low_drone + whispering
   - 战斗时: intense_combat + critical_hit
   - 催泪时: sad_piano + rain
   - 让玩家"听到"情绪
4. **支线对话丰富角色**：
   - 谢临川的潜台词（"我知道你在被排挤"）
   - 同门的排挤（宗门氛围）
   - 黑衣人的™ SEL 暗示（身份伏笔）
5. **节奏把控**：
   - 每段落 ~8-12 分钟，总 45 分钟可玩
   - 战斗/对话/探索交替，避免疲劳
   - 高潮（家谱+黑衣人）连续制造冲击
