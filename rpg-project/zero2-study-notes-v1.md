## 📚 参考资料学习提炼

### 1. Make Better Town Maps: RPG Maker MZ (YouTube)

**链接**: https://www.youtube.com/watch?v=xFSZ-GzjbNI

**学到的 3 条具体方法**:
1. **分层设计**: 先画主干道（主路/中轴），再画次干道（支路），最后布细节（装饰/植被）
2. **地标引导**: 在主路尽头或拐点放置明显的地标建筑（塔/钟楼/祠堂），引导玩家方向
3. **功能分区**: 明确划分入口区、商业区、住宅区、行政/宗教核心区，避免混杂

**属于结构方法**: 1, 2 (道路规划、地标位置)
**属于风格方法**: 3 (分区主题装饰)

**如何落到 zero2**:
- 主路：从山门 → 中轴 → 主殿前庭，一路直通
- 地标：主殿屋顶（翘角 + 匾额）作为视觉终点
- 分区：
  - 入口区：山门 + 守卫石狮 + 公告墙
  - 过渡区：左右对称的练武场/休息亭
  - 核心区：主殿前庭 + 香炉 + 石碑

### 2. Mapping: Towns | Official RPG Maker Blog

**链接**: https://www.rpgmakerweb.com/blog/mapping-towns

**学到的 3 条具体方法**:
1. **现实比例参考**: 街道宽度 = 2-3  tiles，建筑间距 ≥1 tile，避免拥挤
2. **视线通廊**: 沿主路行走时，视线应能穿透到下一个地标（避免被建筑完全遮挡）
3. **功能暗示**: 通过门口摆件（药篓、兵器架、公告板）暗示建筑用途，不需长文本

**属于结构方法**: 1, 2 (比例、视线)
**属于风格方法**: 3 (装饰暗示)

**如何落到 zero2**:
- 主路宽度 = 3 tiles，两侧建筑退让 1 tile 作为缓冲
- 从山门看主殿，中间无高建筑遮挡，仅设矮墙/灯柱作为节奏点
- 主殿门口设香炉+石碑，练武场摆兵器架，公告墙贴任务榜

### 3. RPG Maker MZ: Basics EP-8: How to make a Quest

**链接**: https://www.youtube.com/watch?v=ZLPJ55Iajv8

**学到的 3 条具体方法**:
1. **任务触发器集成**: 任务给予点应放在人流密集处（公告板/NPC 聚集区）
2. **路径标记**: 用地面符号（箭头/光点）或地标（特别颜色的建筑）指示任务目标方向
3. **任务完成反馈**: 完成任务后，给予点应有视觉变化（NPC 表情/灯光），让玩家知道可以交任务

**属于结构方法**: 1, 2 (触发点位置、方向指示)
**属于风格方法**: 3 (视觉反馈)

**如何落到 zero2**:
- 任务给予：公告墙设在入口右侧，玩家必经之路
- 方向指示：主路地面设宗门纹样，指向主殿（任务交付点）
- 完成反馈：主殿门口灯柱完成任务后亮起（或师尊殿窗户亮灯）

### 4. RPG Maker MZ Tutorial #29 - Create Quest Properly!

**链接**: https://www.youtube.com/watch?v=SOhx82x7M3w

**学到的 3 条具体方法**:
1. **任务链空间分布**: 给予、过程、交付三点不宜太近，保持距离感（至少 2-3 个场景）
2. **环境叙事补充**: 在任务路径上放置环境线索（战斗痕迹、破损物品），不靠对话解释
3. **玩家自由度**: 允许玩家绕路探索，但核心路径必须清晰，避免死胡同干扰主线

**属于结构方法**: 1, 3 (分布、死胡同控制)
**属于风格方法**: 2 (环境叙事)

**如何落到 zero2**:
- 任务链：公告墙（给予）→ 练武场/周边（过程）→ 主殿（交付），三点成三角
- 环境线索：练武场沙袋破损、公告墙有涂改、主殿台阶有陈旧血迹
- 自由探索：左右附属区开放，但中轴路径明确，无过多死胡同

### 5. Village Maps for RPG Maker MZ (Casper Gaming)

**链接**: https://casper-gaming.itch.io/village-maps-mz

**学到的 3 条具体方法**:
1. **视觉焦点**: 每张地图应有 1-2 个视觉焦点（喷泉/神龛/大树），吸引玩家目光
2. **道路宽的变化**: 入口窄（2 tiles）→ 主路宽（3-4 tiles）→ 核心区更宽或有广场感，营造递进
3. **边界处理**: 用建筑外墙、树篱、墙壁明确边界，避免开放感（城镇应有封闭感）

**属于结构方法**: 1, 2, 3 (焦点、宽度、边界)
**属于风格方法**: 1 的实现方式 (焦点设计)

**如何落到 zero2**:
- 视觉焦点：主殿前庭的香炉 + 石碑（双焦点），入口石狮
- 道路宽度：山门入口 2 tiles → 中轴 3 tiles → 主殿前广场 5x5 tiles
- 边界：四周用围墙 + 旗杆围合，只留山门一个出口

### 6. Minimalist Chinese Temple tileset (OpenGameArt)

**链接**: https://opengameart.org/content/minimalist-chinese-temple-tileset

**学到的 3 条具体方法**:
1. **屋顶符号化**: 用简化屋檐和翘角表达宗教/宗门感，不需复杂雕花
2. **色彩统一**: 主殿用红/金，辅助建筑用灰/白，形成主次
3. **装饰元素**: 灯笼、香炉、石碑、匾额——4 件套即可定义宗门广场

**属于结构方法**: 无（纯风格）
**属于风格方法**: 1, 2, 3

**如何落到 zero2**:
- 主殿：红墙金瓦 + 翘角，匾额"太玄殿"
- 辅助建筑：灰瓦白墙（练功房、藏书阁）
- 装饰：石狮（入口）、灯柱（中轴两侧）、香炉（主殿前）、石碑（左侧功勋榜）

### 7. Oriental Roof Tiles (OpenGameArt)

**链接**: https://opengameart.org/content/oriental-roof-tiles

**学到的 3 条具体方法**:
1. **屋顶层次**: 歇山顶/庑殿顶用于主殿，硬山顶用于附属，通过屋顶类型区分等级
2. **屋顶曲线**: 翘角轻微上翘，避免平直，增加灵动感
3. **屋顶装饰**: 脊兽（简化版）仅用于主殿，辅助建筑不加

**属于结构方法**: 1 (屋顶类型用于层级)
**属于风格方法**: 2, 3

**如何落到 zero2**:
- 主殿：歇山顶 + 翘角 + 简化脊兽（2个）
- 左右建筑：硬山顶，无脊兽
- 所有屋顶统一色调（深灰瓦+浅灰光面）

## 🎯 Zero2 设计原则总结（从学习引出）

1. **主路必须一眼看出**：中轴 + 宽度递进 + 视线通廊
2. **主殿必须一眼看出**：屋顶等级 + 色彩对比 + 视觉焦点
3. **功能区必须划分**：入口/过渡/核心三区，不混杂
4. **装饰必须有功能**：每件摆饰都应暗示用途或叙事
5. **边界必须封闭**：围墙围合，只留山门出入
