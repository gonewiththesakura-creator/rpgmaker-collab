## 📖 1. 主线推进 #1: 裂谷救援后的选择

**场景**: M004 事件 1 (Rift Edge, after QTE)  
**触发条件**: `GS_RIFT_COLLAPSE=true` 且 QTE 救援完成  
**情绪目标**: 紧张 + 责任（"你的选择有后果"）  
**压缩文本** (MZ 直接贴):

```
你在做什么: 你单手抠着岩缝，另一只手还拉着同伴。
为什么: 因为你选了救他——现在没有退路。
下一步: A. 全力向上(快速按键)  B. 稍作休息(风险:二次塌方)

[如果 QTE 成功]
"值得...你的重量。"
变量: GS_RIFT_CLIMB_SUCCESS=true
```

**为什么必须保留**:
- 这是玩家**第一次主动救援**而非被保护，角色关系转折点
- QTE 提供游戏性，文本明确选择后果（紧张感）
- 变量 `GS_RIFT_CLIMB_SUCCESS` 影响后续谢临川对话
- 缺失会导致裂谷事件变成"纯 QTE 操作"，无情绪收束

## 📖 2. 主线推进 #2: 家谱核心真相 (3 页压缩)

**场景**: M007 事件 1 (Cave Midway, Registry Investigation)  
**触发条件**: 进入家谱路径 (`GS_REGISTRY_PATH_CHOSEN=true`)  
**情绪目标**: 震撼 → 自我怀疑（"这些句子在描述我"）  
**压缩文本** (3 页，每页 ≤3 行):

**第1页**:
```
皮革封面冰冷。"沈氏血脉"泛暗金。
最新页:"沈照璃"墨迹未干——昨晚才写。
你心跳漏一拍。
```

**第2页**:
```
残页: "...九劫归一...六魂归位..."
"...深渊之中，残我沉眠..."
列表: 绮罗音、绯铃、宁素心、白清漪、姬无瑕、原初照璃。

你在做什么: 意识到——这些句子在描述**你**。
```

**第3页**:
```
家谱荧光射向你心脏。壁画亮起:六道身影跪拜，阵心晶体与你玄牝天香体光芒**一模一样**。
壁画新字: "他们都在等你回来。"

身后脚步声。谢临川影子——**多出一道轮廓**。

变量: GS_RESIDUE_IDENTITY_HINT=true, GS_SHADOW_SEEN=true
```

**为什么必须保留**:
- **整个 Chapter 3 的核心谜题揭示**，缺失则玩家不知残我是什么
- "字迹新"、"荧光心脏"、"影子"三层伏笔一次性引爆
- 变量同时触发多系统（Ch4 对话、残我机制）
- 压缩后仍保持悬疑与冲击，每行都有信息量

## 📖 3. 支线情感 #1: 裂谷救援羁绊 (QTE 后谢临川对话)

**场景**: M004 事件 1 后，如果 `GS_XIE_LIFESAVER=true`  
**触发条件**: QTE 成功救出谢临川  
**情绪目标**: 羁绊 + 承诺（"我信任你，所以为你冒险"）  
**压缩文本**:

```
谢临川(喘息):"...为什么？你明明可以自己上去。"

"你说过，练剑场那边没人会打扰。\n现在换我——别想推开我。"

谢临川沉默，手在发抖。
"...你比我想的强。\n记住今天。这份情，我还。"

变量: GS_BOND_WITH_XIE_STRENGTHENED=true
```

**为什么必须保留**:
- **谢临川角色弧光的起点**：从"保护者"到"被保护"的情感转折
- 为 Ch3 他的牺牲独白埋下伏笔（"记得裂谷那次吗？"）
- 2 句话建立深层羁绊，比长独白更 punchy
- 缺失则谢临川变成纯功能角色，无个人情感

---

## 📖 4. 支线情感 #2: 营地篝火对话启动 (三选一)

**场景**: M005 事件 1 (Temporary Camp, night)  
**触发条件**: 进入营地 (`GS_CAMP_RESTED=true`)  
**情绪目标**: 温暖 + 抉择（"短暂安宁中决定听谁的故事"）  
**压缩文本**:

```
篝火噼啪。三个人: 老兵(抽烟)、药剂师(草药)、谢临川(擦剑)。

你在做什么: 决定先问谁——时间有限。
为什么: 每人知道一部分真相。
下一步:
  A. 问老兵("沈家军覆灭?")
  B. 问药剂师("青鸾未死?")
  C. 问谢临川("你的承诺?")
  D. 冥想(恢复 MP)

变量: GS_CAMP_FIRST_CHOICE=1/2/3/4
```

**为什么必须保留**:
- **唯一能让玩家"决定听谁故事"的自由度**，提升代入感
- 三条支线并行，根据选择获得不同奖励/信息
- 为 Old Sanctuary、药剂师青鸾、谢临川承诺三条支线提供入口
- 缺失则营地变成"自动播放信息"，无玩家 agency

## 📖 5. 世界观线索 #1: 秘洞三岔门选择

**场景**: M006 事件 1 (Cave Entrance)  
**触发条件**: 抵达秘洞入口  
**情绪目标**: 悬念 + 紧迫（"三条路，必须选"）  
**压缩文本**:

```
三扇门:
左: 沈家徽(⇒ 家谱)
中: 残我纹(⇒ 试炼)
右: 素问印(⇒ 补给)

你在做什么: 选择优先级。
为什么: 你无法全都要。
下一步: A(左) B(中) C(右) D(设陷阱)

变量: GS_REGISTRY_PATH_CHOSEN=true 等
```

**为什么必须保留**:
- **首个重大分支选择**，让玩家思考"我要什么"
- 三条路径对应三条内容线（家谱主线/残我支线/物资支线）
- 选择直接影响后续 10 分钟体验
- 缺失则秘洞变成单线程，失去探索感

## 📖 6. 世界观线索 #2: 家谱揭示六残我

**场景**: M007 事件 1 第 2 页 (见上方主线推进 #2)  
**触发条件**: 家谱阅读第 2 页  
**情绪目标**: 认知颠覆（"我竟是其中之一"）  
**压缩文本** (第 2 页核心句):

```
列表: 绮罗音、绯铃、宁素心、白清漪、姬无瑕、原初照璃。

你在做什么: 意识到——这些句子在描述**你**。
变量: GS_RESIDUE_IDENTITY_HINT=true
```

**为什么必须保留**:
- **"六残我"概念的唯一正式揭示**，缺失则玩家不理解残我机制
- 7 个字"这些句子在描述**你**"是镜中时刻的精华
- `GS_RESIDUE_IDENTITY_HINT` 是 Ch4 多个对话的前提条件
- 这是 Chapter 3 最大的"啊哈！"时刻，必须保留

## 📖 7. 收束/钩子 #1: 家谱完成离场抉择

**场景**: M008 事件 1 (Cave Depths, after Registry)  
**触发条件**: 完成 3 页家谱阅读  
**情绪目标**: 紧迫 + 责任（"黑衣人可能在来，快决定"）  
**压缩文本**:

```
你合上家谱，手在抖。

下一步:
  A. 立即返回营地(告诉同伴)
  B. 继续探索秘洞(可能更多线索)
  C. 隐藏家谱(谁都不说)

"黑衣人可能在附近——快决定。"
变量: GS_REGISTRY_FULLY_ANALYZED=true
```

**为什么必须保留**:
- **Chapter 3 结尾的最后一个选择**，直接影响 Ch4 开场状态
- 三条选项各有后果（A: 团队知情；B: 个人深入；C: 秘密压力）
- "黑衣人可能在附近"制造离场张力，避免玩家在秘洞无限停留
- 缺失则章节结束突兀，无过渡到 Ch3 高潮

---

## 📖 8. 收束/钩子 #2: 黑衣人逼近预警

**场景**: M008 事件 3 (Cave Depths, near exit)  
**触发条件**: 玩家接近秘洞出口（无论是否完成家谱）  
**情绪目标**: 恐惧 + 逃亡驱动（"快走，他来了"）  
**压缩文本**:

```
远处传来脚步声——沉重、有节奏。
不是同伴。

"黑衣人。他在搜，快走！"
画面边缘变暗，紧张音效渐强。

变量: GS_BLACKCLOAK_APPROACHING=true
```

**为什么必须保留**:
- **Chapter 3 开场的延续**：直接触发 Ch3-1 黑衣人突袭
- 提供时间压力，让玩家"带着紧张离开秘洞"
- 音效+画面变暗创造沉浸感，不仅是文字提示
- 缺失则 Ch3 开始太突然，缺少铺垫
