# Writer Playtest Feedback Capture - Phase 3 RC v1

**Purpose**: Structured template for capturing real player feedback during TP-06~TP-10 playtest
**Status**: Stand by - awaiting QA / owner playtest results
**Scope**: Only record what players actually say and do, no premature analysis

---

## 📋 反馈收集总原则

1. **客观记录**: 只记玩家原话、行为、停留时间，不做主观解读
2. **对应测试点**: 每条反馈关联到具体 TP-01~TP-10
3. **情绪标记**: 用 😊/😲/😢/😠/🤔 快速标注瞬间反应
4. **可复现**: 记录触发条件（如 `GS_XIE_LIFESAVER=true`）以便复现问题
