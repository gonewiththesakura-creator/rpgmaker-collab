**Status**: Ready for Tech Integration

## 📖 扩展 1: 裂谷崩塌后的 "你在做什么/为什么/下一步"

### 场景: M009 裂谷边缘 (QTE 救援后)

**你在做什么**
> 你单手抓着崖壁，另一只手还拉着谢临川（或另一位同伴）。碎石继续滚落深渊，你的手臂快麻木了。

**为什么这么做**
> 因为刚才的瞬间选择——你选择了救他。现在没有退路：要么一起爬上去，要么一起掉下去。
> 更重要的是，**他信任你才松手**（如果他喊了"别管我"），你不愿辜负这份信任。

**下一步是什么**
> - A. 用尽全力向上攀爬（QTE 再次触发）
> - B. 寻找落脚点稍作休息（风险：碎石可能再次塌方）
> - C. 让同伴先上去（如果同伴状态较好）

**变量触发**:
- 选择 A 且 QTE 成功 → `GS_RIFT_CLIMB_SUCCESS = true`
- 选择 B → `GS_RIFT_REST_ATTEMPT = true`（可能触发二次塌方事件）
- 选择 C → `GS_PARTNER_FIRST_UP = true`（同伴上去了但可能不回来救你）

---

### 情绪收束
- **成功爬上去**: 脱力瘫倒，但同伴 expressions of gratitude → 羁绊加深
- **失败/二次塌方**: 强制 readload，但保留"你选择了救"的变量（不影响主线，但影响支线评价）

## 📖 扩展 2: 营地篝火 "你在做什么/为什么/下一步"

### 场景: M010 营地夜晚

**你在做什么**
> 你坐在篝火旁，手里握着刚获得的 Hot Token（符纹），它像心跳一样微微搏动。
> 周围有三个人：老兵（沉默抽烟）、药剂师（整理草药）、谢临川（擦拭剑）。

**为什么这么做**
> 因为你需要**决定先和谁深谈**。每个人都知道一部分真相：
> - 老兵知道沈家军历史
> - 药剂师知道残我线索
> - 谢临川知道自己的身份和承诺
>
> 时间有限，黎明前必须出发去秘洞。

**下一步是什么**
> - A. 先问老兵："沈家军为什么覆灭？"
> - B. 先问药剂师："青鸾未死是什么意思？"
> - C. 先问谢临川："你说的承诺是什么？"
> - D. 谁也不问，独自冥想（恢复 MP，可能触发残我低语）

**变量记录**:
- `GS_CAMP_FIRST_CHOICE = 1/2/3/4`（记录优先顺序）
- 如果全部对话过 → `GS_CAMP_FULL_DIALOGUE = true`

---

### 情绪设计
- **篝火环境**: 安全但短暂，营造"最后喘息"的珍贵感
- **对话顺序**: 影响后续角色对你的态度（老兵如果先谈，会更信任你；谢临川如果最后谈，会透露更多）
- **奖励触发**:
  - 老兵故事听完 → `Hot Token` 升级为 `Friendly Token`（战斗中自动吸引沈家旧部仇恨）
  - 药剂师说完 → `Three-Leaf Reviving Herb` 数量+2
  - 谢临川说完 → `GS_XIE_PROMISE_FULL = true`（Ch3 独白增加 20 字）

## 📖 扩展 3: 秘洞中段 "你在做什么/为什么/下一步"

### 场景: M016 秘洞 midway (石室迷宫)

**你在做什么**
> 你站在三岔路口，每道门有不同符号：
> - 左门：沈家徽记（⇒ 家谱）
> - 中门：残我纹样（⇒ 陷阱/试炼）
> - 右门：素问真人印记（⇒ 补给/信息）

**为什么这么做**
> 因为你知道时间不多（黑衣人的线索显示他也来了）。
> 但你也需要：
> - 家谱（主线必得）
> - 残我试炼（了解自己）
> - 素问的指引（她可能留下信息）
>
> **你不能全都要**，必须优先级排序。

**下一步是什么**
> - A. 左门（家谱）→ 触发家谱事件链
> - B. 中门（残我试炼）→ 可选战斗，获得 `GS_RESIDUE_FRAGMENT_HINT`
> - C. 右门（素问印记）→ 获得 `Qingluan's Note`（纸条："照璃，我在深渊等你"）
> - D. 设陷阱/标记（如果 `GS_SOLDIER_FATHER_MENTION = true`，老兵教你在门上刻沈家暗号，防止黑衣人跟入）

**变量**:
- 选择 A → `GS_REGISTRY_PATH_CHOSEN = true`
- 选择 B → `GS_RESIDUE_TRIAL_COMPLETED = true`（+ 碎片 hint）
- 选择 C → `GS_QINGLUAN_NOTE_OBTAINED = true`
- 选择 D → `GS_TRAP_SET = true`（可能延迟黑衣人 2 回合）

---

### 情绪设计
- **紧迫感**: 黑衣人可能在后面，选择压力大
- **后果可见**: 无论选哪个，都会在其他门触发"错过的线索"提示（如选 A 后，B、C 门锁死，但门上符号被记住）
- **变量反馈**: Ch4 如果 `GS_QINGLUAN_NOTE_OBTAINED = true`，素问会说："你看到了我的字条...我等你。"

---

## 📖 扩展 4: 战斗结束后的内心独白（收束情绪）

### 触发时机: 任何战斗胜利/失败后，如果 `GS_CAMP_FULL_DIALOGUE = true`

**独白模板**（根据战斗结果和之前营地对话动态生成）

> **如果胜利**:
> "这场战斗让我明白，老兵说的'沈家军精神'不是口号。每一个倒下的人，都曾是和眼前一样的血肉之躯。Hot Token 在我手中发烫，像是在为他们复仇。"
> 
> **如果失败但逃生**:
> "我们逃出来了，但谢临川的话还在耳边——'必须失去一些东西'。这一战，我们失去了什么？是装备，还是信心？还是...对这个世界最后的信任？"
> 
> **如果谢临川在队且重伤**:
> "看着谢临川的伤口，我忽然想起营地药剂师的话：'青鸾未死'。素问真人还活着，对吗？那么沈家的希望也还在。我必须走下去，为了那些说过'等你回来'的人。"
> 
> **老兵在队且 fired**:
> "老兵冲在我前面，他的背影让我想起父亲。Hot Token 的光芒在他身上闪现——这是同心纹的力量。原来，沈家从未真正覆灭。"

**变量触发**: `GS_POST_COMBAT_REFLECTION = true`（标记已触发）

---

### 设计意图
- **情绪收束**: 战斗不只是数字变化，要有心理后果
- **串联支线**: 将老兵、药剂师、谢临川的线索在战斗后整合，强化"他们的故事与我有关"
- **变量系统**: `GS_POST_COMBAT_REFLECTION` 确保每个战斗后只触发一次，避免重复

## 📖 扩展 5: Old Sanctuary Shrine 支线完成（最终情绪对话）

### 支线背景
- **触发**: 如果玩家在营地听了老兵完整故事 (`GS_SOLDIER_FULL_STORY = true`) 且在裂谷带了 Hot Token
- **地点**: M010-2 Old Sanctuary Shrine（营地旁的废弃祠堂）
- **完成条件**: 在祠堂祭拜，放置 Hot Token 于沈家牌位前

### 最终情绪对话

**你在做什么**
> 你在沈家祠堂前，将 Hot Token 放在牌位前，点燃三炷香（自动从背包使用）。

**为什么这么做**
> 因为老兵告诉你这里曾是沈家军祭祖之地，而你的符纹是"同心纹"——只有真正的沈家血脉才能让它真正共鸣。
> 你想知道，**符纹是否能唤醒什么**。

**下一步**
> 系统自动触发序列：
> 1. 香燃起，祠堂内光影晃动
> 2. Hot Token 浮空，纹路流动，显示所有阵亡者的名字（数百个）
> 3. 老兵的回忆独白（从对话框中自动播放）

**独白内容**:
> 老兵（低声，哽咽）:"这些名字...有些是我兄弟，有些是我儿子。我们为沈家流过血，但今天，**你的血也在其中**。"
> 
> 他停顿，看着符纹光芒照在你身上:
> "令牌选择你，不是因为你姓沈，而是因为**你的心**。你愿意救陌生人，愿意直面残我——这才是沈家军的精神。"
> 
> "拿着。祠堂地底有最后一份补给，是我藏的五雷符和回复药。拿去吧，接下来的路，更需要。"
> 
> **物品奖励**: `Five-Thunder Talisman` x3（强力攻击道具）, `Elixir` x2
> **变量**: `GS_OLD_SANCTUARY_COMPLETED = true`, `GS_SOLDIER_RESPECT_EARNED = true`

**后续影响**:
- 如果 `GS_SOLDIER_RESPECT_EARNED = true`，在 Ch4 老兵会托人带话："符纹在你手里跳动，战场见，孩子。"
- `Five-Thunder Talisman` 在 Ch3 黑衣人战斗中有特殊对话：" veterans' power!"

## 📖 扩展 6: Fallen Scout 支线完成（最终情绪对话）

### 支线背景
- **触发**: 如果玩家在入山时选择了独行 (`GS_TRIAL_ENTERED_WAY_GROUP = false`) 或在裂谷展示了勇气 (`GS_XIE_LIFESAVER = true`)
- **地点**: M008-2 悬崖边隐蔽处（需要调查发现）
- **发现**: 一具 Scout 的尸体，身边有破损的沈家地图和半封信

### 最终情绪对话

**你在做什么**
> 你收起 Scout 的信和地图，突然，尸体上的玉佩（和你的一模一样）发出微弱共鸣。

**为什么这么做**
> 因为地图标记了秘洞入口和一条**安全隐蔽路径**，能避开黑衣人可能的伏击。
> 但更让你在意的是——**这个 Scout 为什么会有沈家玉佩？** 他到底是什么身份？

**下一步**
> 系统触发:
> 1. 玉佩共鸣加强，一段记忆闪回（画面：Scout 在暴雨中交给你父亲一个包袱，说"我会保护她"）
> 2. 你意识到——这个 Scout 是**十年前派来暗中保护你的**，直到你也进入试炼

**独白/对话**（如果谢临川在队）:
> 你问谢临川是否知道 Scout。
> 谢临川（第一次露出悲伤）:"我父亲...也是暗卫。他说过，真正的守护者，从不留名。"
> 
> "我们是™ SEL 的最后一支，用来保护沈家的火种。现在...火种已经足够强，不需要我们再出现了。"
> 
> 他拍拍你的肩:"带着他们的信念，走下去。"

**变量**: `GS_SCOUT_IDENTITY_REVEALED = true`, `GS_HIDDEN_PROTECTORS_KNOWN = true`
**物品升级**: 你的玉佩 become ` Guardian's Pendant`（防御+10，且自动记录所有沈家旧部位置）

**后续影响**:
- Ch4 如果 `GS_HIDDEN_PROTECTORS_KNOWN = true`，素问会说："沈家的暗卫...他们还活着，只是你已经不需要他们了。"
- 残我对话中，绮罗音可能提及："那些守护者，终于完成了他们的使命。"

## 🎯 整体体验设计（Phase 3 完成版）

**总时长**: ~60 分钟 (Line B) + 20 分钟支线 = 80 分钟深度体验

**核心循环**:
```
选择（入山路线） → 危机（裂谷崩塌） → 喘息（营地对话） → 选择（秘洞分支） → 高潮（家谱调查）
   ↓支线插入
旧祠堂祭拜（老兵 respect） / 悬崖发现（Scout 身份） → 奖励/信息 → 情绪收束（战斗独白）
```

**变量系统完整度**:
- 战术变量（战斗、资源）
- 叙事变量（信任、了解度）
- 支线变量（Old Sanctuary, Fallen Scout）
- 所有变量在 Ch4 有回声，确保"选择有意义"
