# ASSET_PLACEMENT_CHECKLIST_v1

适用对象：A/C
用途：快速检查“地图看起来像不像游戏”。

## A. 结构资产（先放）
### 宗门图（M001）
- [ ] 建筑 2~3
- [ ] 中心地标 1（石碑/法阵）
- [ ] 路灯/石灯 4+
- [ ] 石板路连通关键NPC
- [ ] 栅栏/墙体形成边界

### 周边图（M002）
- [ ] 主路 + 1 条支路
- [ ] 断墙/遗迹 2+
- [ ] 树木 8+
- [ ] 草丛/小石 10+
- [ ] 路标 1~2

### 古碑图（M003）
- [ ] 古碑主地标 1
- [ ] 战斗区边界清晰
- [ ] 裂痕/碎石 8+
- [ ] 枯木/残柱 3+
- [ ] 冷色调氛围设置

## B. 互动资产（再放）
- [ ] 每图调查点 >= 3
- [ ] 调查点周围有“视觉提示”
- [ ] 关键道具点位不隐形
- [ ] 触发点不与传送点重叠

## C. 氛围资产（最后放）
- [ ] M001 暖色/稳定
- [ ] M002 过渡色调
- [ ] M003 冷色+雾效
- [ ] 战前压暗/停顿

## D. 可玩性红线
- [ ] 主路径不会卡死
- [ ] 不会误触关键剧情
- [ ] 事件后仍可回退到安全点
- [ ] 必败后退场逻辑不被地形阻塞
