## 步骤2: M001宗门广场 (40分钟)

### 2.1 创建地图

**操作**: 右键地图列表 → 新建 → 地图设置

**设置**:
- 名称: M001宗门广场
- 尺寸: 宽20，高15
- Tileset: Overworld (或 Exterior)
- BGM: 选择平和的BGM (如Town1)

### 2.2 绘制地图

**使用Tileset绘制**:
- 中央区域(5,5)-(15,10): 铺设广场地面
- 边缘: 铺设建筑/墙壁作为边界
- (10,14): 下方出口（用传送点标记）

### 2.3 创建事件1: E101开场初始化

**位置**: (0,0) 或其他角落

**事件页设置**:
- 触发条件: 自动执行

**事件命令** (按顺序添加):

```
1. 显示文本 ["", 0, 0, 2]
   - 文本1: 【回忆】
   - 文本2: 沈家...灭门...
   - 文本3: 你发誓要找出真相。

2. 独立开关操作 [A, ON]

3. 开关操作 [1, ON]  (system_tutorial_done)

4. 场所移动
   - 直接指定: M001, X:10, Y:8
   - 方向: 保留
   - 淡入淡出: 无
```

### 2.4 创建事件2: E102素问真人

**位置**: (10, 5)

**图像**: 选择角色图 (如Actor1-0)

**事件页设置**:
- 触发条件: 确定键
- 优先级: 与人物相同

**事件命令**:

```
1. 显示文本 ["Actor1", 0, 0, 2]
   - 文本1: 素问真人：欢迎来到青云宗。
   - 文本2: 你体质特殊，需多加修炼。

2. 独立开关操作 [A, ON]

3. 开关操作 [10, ON]  (plot_ch1_done)
```

**事件页2** (新建页):
- 条件: 独立开关A=ON
- 内容: 显示文本"素问真人：加油修炼。"

### 2.5 创建事件3: E104谢临川

**位置**: (15, 8)

**图像**: 选择角色图 (如Actor2-1)

**事件页1**:
- 触发条件: 确定键

## 步骤4: M003古碑区域 (50分钟)

### 4.1 创建地图

**新建地图**:
- 名称: M003古碑区域
- 尺寸: 宽20，高20
- Tileset: Dungeon (或 Cave)
- BGM: Dungeon3 (压迫感)
- BGS: Wind (风声)

### 4.2 绘制地图

- 入口(1,1): 狭窄峡谷
- 中央(10,10): 开阔区域，放置古碑
- 使用暗色tile营造氛围

### 4.3 创建古碑事件 (E301+E302)

**位置**: (10, 10)

**图像**: 选择图标 !Crystal

**事件页1**:
- 触发: 确定键
- 图像: 发光效果 (Step Animation: ON)

**事件命令**:

```
1. 显示文本 ["", 0, 0, 2]
   - 发现一块古老的石碑...

2. 画面色调 [-68,-68,-68,0], 30帧, 等待

3. 显示文本 ["", 0, 0, 2]
   - 【古碑文字】
   - 九劫归一...
   - 沈家血脉...

4. 画面色调 [0,0,0,0], 30帧, 等待

5. 独立开关 [A, ON]
```

**事件页2**:
- 条件: 独立开关A=ON
- 内容: 显示文本"已经调查过了。"

### 4.4 创建黑衣人触发事件 (E303)

**位置**: (0,0) 或任意位置

**触发条件**: 并行处理

**事件命令**:

```
1. 条件分支: 玩家位置 X >= 8 且 Y >= 8 (接近古碑)
   - 画面色调 [-100,-100,-100,0], 30帧, 等待
   
   - 显示文本 ["", 0, 0, 2]
     - 黑衣人：把东西交出来！
   
   - 调用公共事件 [1]  (黑衣人战斗)
```

### 4.5 创建公共事件: 黑衣人战斗

**操作**: 游戏 → 数据库 → 公共事件

**公共事件1**:
- 名称: 黑衣人战斗
- 触发: 无

**事件命令**:

```
1. 战斗处理
   - Troop: 1 (黑衣人)
   - 可以逃跑: NO
   - 失败继续: YES
```

### 4.6 创建Troop: 黑衣人

**操作**: 游戏 → 数据库 → Troop

**Troop 1**:
- 名称: 黑衣人
- 成员: 敌人1号 x2 (使用默认敌人或新建)

**Troop Event** (关键!):
- 页面: 1
- 触发: 回合结束
- 范围: 整个队伍
- 条件: 无 (每回合都检查)

**事件命令**:

```
条件分支: 脚本
  - 代码: $gameParty.members()[0].hp <= $gameParty.members()[0].mhp * 0.3
  - 或: BattleManager._turnCount >= 5
  
  - 中断战斗
  - 调用公共事件 [2]  (败退获取家谱)
```

**脚本代码** (在条件分支的脚本中输入):
```javascript
$gameParty.members()[0].hp <= $gameParty.members()[0].mhp * 0.3 || BattleManager._turnCount >= 5
```

### 4.7 创建公共事件2: 败退获取家谱

**公共事件2**:
- 名称: 败退获取家谱

**事件命令**:

```
1. 显示文本 ["", 0, 0, 2]
   - 你支撑不住了...
   - 黑衣人逼近...
   - 突然，一道剑光闪过！
   - 黑衣人撤退了。

2. 显示文本 ["", 0, 0, 2]
   - 在混乱中，你发现了一块残破的玉简...

3. 增减物品 [残破家谱, +1]

4. 显示文本 ["", 0, 0, 2]
   - 获得【残破家谱】！

5. 开关操作 [12, ON]  (plot_ch3_done)

6. 场所移动 → M001 (10, 10)
```
