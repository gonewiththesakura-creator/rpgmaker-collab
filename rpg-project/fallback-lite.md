## 裁剪优先级总览（P1→P5）

| 优先级 | 策略 | 预计节省 | 影响评估 | 可恢复性 |
|--------|------|---------|---------|---------|
| **P1** | Ch1 教学事件合并 | 5-8 事件 | 新手引导简化，可摸索 | 高 |
| **P2** | Ch1 同门排挤压缩 | 2 事件 | 宗门竞争氛围减弱 | 中 |

**总计可节省**：16-23 事件 → 最终 246-279 事件，低于 350 安全线 ✅

## P1: Ch1 教学事件合并（最安全）

### 裁剪内容
- M001-1 宗门入口（移动教学） + M001-2 宗门广场（对话教学） → 合并为 1 个场景

### 实施方法
```mz
// 原 M001-1：移动教学后自动进入广场
// 优化：移动到广场后自动触发对话教学，减少独立移动教学事件

// 修改后：
Scene 1-1 "宗门入口" 
- 触发：游戏开始
- 步骤：玄微子对话 → "去广场找谢临川" → 玩家移动至广场（小地图指引） → 到达后自动触发谢临川对话（同时教学对话交互）
```

### 文案影响
- 玄微子台词增加："去吧，谢临川会教你如何与 NPC 交谈。"
- 谢临川初遇台词后增加："看，很简单吧？"（教学确认）

### 节省：5-8 事件（原 M001-1 8 事件 + M001-2 部分重叠）

## P2: Ch1 同门排挤压缩（氛围次要）

### 裁剪内容
- 原 2 个同门事件（同门A、同门B）→ 合并为 1 个"宗门氛围"事件

### 实施方法
```mz
// 原设计：同门A（排挤） + 同门B（竞争） = 2个独立Event
// 优化：合并为 Scene 1-2 "宗门广场"中的一个"多NPC对话"事件

// 修改后：
事件"宗门氛围"
- 触发：与谢临川对话后
- 内容：同时显示同门A/B的简短对话（通过"对话选项"选择听谁说）
- 选项：听A（排挤）/听B（竞争）/都不听（直接离开）
```

### 文案影响（合并后）
```
同门A："沈家遗孤？别靠太近。"
同门B："新人，宗门只看实力。"
```
从 2 事件压缩为 1 事件（多选项，但仅触发一个显示事件）。

### 节省：2 事件（原各 3-5 事件 → 合并后 1 事件）

## P3: Ch2 试炼支路缩减（战斗多样性次要）

### 裁剪内容
- 试炼秘境中的 3 波敌人 → 压缩为 2 波，但增强单波强度

### 实施方法
```mz
// 原设计：
波次1：史莱姆x3（简单）
波次2：妖狐x2（中等）
波次3：妖兽x1（Boss）

// 优化：
波次1：史莱姆x2 + 妖狐x1（混合）
波次2：妖兽x1（强化版，HP+20%）
```

### 文案影响
- 删除波次1的"轻松取胜"描述
- 波次2战斗前增加提示："这头妖兽比普通更强，小心！"
- 怪物掉落调整：经验值不变，物品掉落率提升 10%（补偿玩家）

### 节省：3-5 事件（原 3 个独立战斗事件 → 2 个）

## P4: Ch3 古碑谜题简化（解谜深度次要）

### 裁剪内容
- 古碑外围封印解谜从 3 步骤 → 1 步直接开启

### 实施方法
```mz
// 原设计：
步骤1：阅读碑文（提示五行）
步骤2：找到散落的3个符文石
步骤3：按顺序嵌入石碑

// 优化：
玩家调查石碑直接触发："玉佩自动共鸣，石碑缓缓移开。"
（删除中间步骤，但保留"九劫归一"碑文阅读）
```

### 文案影响
- 删除"符文石"物品
- 石碑调查文案增加："玉佩与凹槽吻合，嵌入后石门开启。"
- 保持谜题的"仪式感"但去掉操作步骤

### 节省：3 事件（原 1-2-3 步骤事件 → 1 个调查事件）

## P5: Ch4 深渊移动压缩（氛围次要）

### 裁剪内容
- 深渊路径长度减少 30%，行走事件从 15 步 → 10 步

### 实施方法
```mz
// 原设计：长通道 + 多个"前进/返回"事件
// 优化：设置"自动移动"事件，玩家按下方向键后自动走到下一场景

// 技术：使用"事件跳跃"或"传送+移动动画"
事件：
- 玩家进入深渊入口
- 自动播放移动动画（5秒）→ 直接到达镜面空间
- 中间减少2个中间区域事件
```

### 文案影响
- 去掉沿途的环境描述事件（"墙壁上的符文"等）
- 保留入口处"色调变暗+速度降低"的总体氛围
- 镜面空间登场文案加强补偿："一阵恍惚，你已站在镜面中央。"

### 节省：3-5 事件（原路径事件 + 中间区域）

## 裁剪决策树（当事件数超标时）

```
if (总事件 > 380) {
  启用 P1（教学合并）→ 节省 8
} if (总事件 > 370) {
  启用 P2（同门压缩）→ 节省 2
} if (总事件 > 360) {
  启用 P3（试炼缩减）→ 节省 4
} if (总事件 > 350) {
  启用 P4（谜题简化）→ 节省 3
} if (总事件 > 340) {
  启用 P5（移动压缩）→ 节省 4
}
最终底线：总事件 ≥ 330（核心 P0 事件 262 + 最低填充 68）
```

- 残我遭遇与选择（M028-M030）
- 所有章节结尾钩子事件
