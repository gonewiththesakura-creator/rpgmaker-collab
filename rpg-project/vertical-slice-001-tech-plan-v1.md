## 4. 谢临川"剧情同行方案B"具体实现口径

### 核心机制确认
**方案B：剧情同行（事件跟随）**  
不加入真实队伍，通过事件动画和对话表现"同行"状态。

### 技术实现细节

#### 1. 同行状态控制
```
开关 S[101] xie_following 控制同行状态

触发条件：
- Ch2 E202：师尊发放试炼后自动开启
- Ch3：保持开启（谢临川参与古碑探索）
- Ch4 E403：深渊中坚持同行，重新确认

关闭时机：
- 章节结束时自动关闭
- 特定剧情点暂时离队（如有需要）
```

#### 2. 事件跟随表现
```
方式：固定位置触发点（推荐，低成本）

实现：
- 在关键地图设置谢临川NPC事件
- 条件：S[101]=ON时显示，OFF时隐藏
- 玩家接近时自动触发对话/协作动画

示例（试炼秘境M004）：
- 位置：秘境入口、关键岔路、终点前
- 事件：提示、协作动画、战斗支援
```

#### 3. 战斗支援模拟
```
公共事件：Common Event 001 "谢临川支援"

触发：Troop Event - Battle Start
条件：S[101]=ON 且 当前地图为试炼/战斗地图

效果：
1. Show Text："谢临川的剑光一闪！"
2. Change State：玩家获得"支援"状态（攻击+20%，3回合）
3. 或 Recover HP：恢复10%HP

实现命令：
- Conditional Branch：S[101]=ON
- Show Text
- Change State / Recover HP
```

#### 4. 情劫进度触发
```
V[003] xie_affection 控制情劫节点

0：初遇（Ch1）
20：试炼协作（Ch2 E204）
50：古碑接应（Ch3 E306）
80：深渊坚持（Ch4 E403）

触发事件：
- 特定地点 + 变量值达到 + 开关条件
- 显示对应情感对话（Writer提供文案）
```

### 方案B优势确认
- 开发成本低（无需AI、升级、装备系统）
- 叙事控制强（精确控制出现时机）
- 维护简单（集中管理，易于修改）
- 符合监督"低成本边界"要求

#### 定位
- 表层反派，服务于Ch3高潮
- 正面对抗，获得残破家谱
