# 太玄界母企划监督摘要

Last updated: 2026-03-18

## 一句话定位
《太玄界》是一款**多周目修仙剧情 RPG**：表层是大女主成长、宗门修行、复仇与正魔分歧，深层是残我回收、轮回破局、九劫归一与真仙登天。

## 项目真正的核心卖点
1. **多周目不是重复通关，而是逐步回收“失败过的自己”**。
2. **残我系统**不是单纯 Boss 设计，而是叙事核心。
3. **结局不是奖励 CG，而是下一周目的遗钥**。
4. **最终成仙不是变得更强，而是成为完整**。

## 世界观核心
- 世界名：太玄界
- 表层问题：灵机衰退，人人修仙、无人飞升
- 深层真相：仙路并非自然断绝，而是被篡改和封锁
- 真正门槛：不是修为不够，而是“人并不完整”

## 女主核心设定
- 主角：沈照璃
- 出身：沈家遗孤，幼年灭门
- 天赋：玄牝天香体、月阴灵根、神识异脉
- 核心命题：她不只是天才，而是能触碰仙路真相、回收诸残我并最终完成归一的人

## 关键人物结构
### 锚点角色
- 谢临川：人间锚点、现实连接、情感重心
- 素问真人：救命恩人与最早意识到主角异常的人

### 残我系角色
- 绮罗音：欲念残我
- 绯铃：玩心残我
- 宁素心：压抑残我
- 白清漪：初心残我
- 姬无瑕：伪仙残我
- 原初照璃：最初真我

### 外部镜像角色
- 涂瑶
- 寒见雪

## 系统核心
### 残我 / 深渊
- 深渊是飞升失败者执念、残魂、道痕沉积之地
- 关键 Boss 本质上是主角前世失败轮回中的残我
- 击败不是结束，净化、收编、归位才是关键机制

### 遗钥系统
不同结局会留下遗钥，成为后续周目推进真相的必要凭证。

关键遗钥包括：
- 无垢心印
- 天香堕印
- 镜欲双生佩
- 断情寒魄
- 红尘留痕
- 伪仙残蜕
- 原初心灯
- 归一命火

### 九劫归一阵
- 前期是伪开阵
- 后期显露真相
- 真作用不是吞材料，而是要求主角归还失落之我

## 主线结构
### 第一阶段：血仇与觉醒
- 宗门日常
- 试炼
- 古碑异变
- 沈家线索
- 情劫与体质初醒

### 第二阶段：分道、复仇、登天
- 正道 / 魔道 / 暗欲三路线
- 九曜门
- 深渊
- 司命殿
- 伪飞升终局

### 真相层
- 多周目揭露残我真相
- 原初照璃 / 姬无瑕 / 沈家亡魂线
- 九劫归一真结局

## 周目结构判断
### 一周目
- 玩家以为这是普通修仙复仇 RPG

### 二周目
- 察觉世界不对劲

### 三周目
- 明确 Boss 与主角同源

### 四周目前置
- 真相逐渐拼完整

### 真周目
- 完成残我归位与真仙路线

## 监督视角下最重要的开发原则
1. **第一阶段必须先把“普通修仙 RPG”表层体验做扎实**，否则真相层没有反差。
2. **残我机制要尽早埋伏笔，但不能太早说破。**
3. **谢临川和素问真人必须成立**，否则人间锚点和情感重心不足。
4. **一周目必须可独立成立为一个完整故事**，不能全靠多周目补救。
5. **技术实现必须控制复杂度**，优先把第一章到第四章打成可玩的 vertical slice。

## 当前最优先的监督任务
1. 定一句官方项目主概念
2. 定第一章到第四章的可制作版本
3. 定第一批关键角色的人设优先级
4. 定 vertical slice 的系统边界（先实现什么，不实现什么）
