### 文件2-4: Map001.json / Map002.json / Map003.json

由于JSON文件较大，详见操作手册的"逐命令操作"部分。

关键事件代码摘要:

**Map001 关键事件**:
- 事件1 (E101): 开场自动播放，初始化变量，传送
- 事件2 (E102): 素问NPC，对话，标记Ch1完成
- 事件3 (E104): 谢临川NPC，对话，S[100]=ON
- 事件4 (E201): 师尊NPC，发布任务
- 事件5: 传送点到M002

**Map002 关键事件**:
- 事件1: 传送回M001
- 事件2: 传送到M003

**Map003 关键事件**:
- 事件1: 传送回M002
- 事件2 (E301+E302): 古碑调查，Tint效果
- 事件3 (E303): 黑衣人出现，调用Common Event

---

### 文件5: data/CommonEvents.json

**公共事件1: 黑衣人战斗**
```
代码: 301 (Battle Processing)
参数: Troop 1, 不能逃跑
```

**公共事件2: 败退获取家谱**
```
代码序列:
- 101+401 (Show Text): 败退剧情
- 126 (Change Items): +残破家谱
- 121 (Control Switches): S[12]=ON
- 201 (Transfer Player): 回M001
```

---

### 文件6: data/Troops.json

**Troop 1: 黑衣人**
- 成员: 敌人ID 1 (x2)
- **Troop Event**: 并行处理，每回合结束
- **条件**: 玩家HP < 30% 或 回合数 >= 5
- **动作**: Abort Battle + 调用Common Event 2

## 3. 最小可运行流程验证步骤

### 验证清单

1. **新游戏开始**
   - 自动播放E101
   - 变量初始化正确

2. **M001流程**
   - 可与素问对话(E102)
   - 可与谢临川对话(E104)，S[100]=ON
   - 可与师尊对话(E201)
   - 可传送到M002

3. **M002流程**
   - 可从M001进入
   - 可传送到M003

4. **M003流程**
   - 可调查古碑(E301+E302)
   - 接近古碑触发黑衣人(E303)
   - 进入战斗
   - 5回合或HP<30%触发败退
   - 败退后获得家谱
   - 传送回M001

### 成功标准
- [ ] 5分钟内可完成全流程
- [ ] 必败触发稳定
- [ ] 败退后流程不断
- [ ] 家谱必得
