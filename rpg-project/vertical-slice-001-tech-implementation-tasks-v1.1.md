## 5. 谢临川方案B可执行步骤

### 核心开关触发点

```

[触发点1: Ch2开始]

条件: E201完成
操作: S[101]=ON
效果: 谢临川开始在特定地图出现

[保持期: Ch2-Ch3]
期间: S[101]保持ON
表现: M004、M005、M006特定位置有谢临川NPC

[触发点2: Ch4坚持]
```

```
- 条件: 无
- 内容: 空（S[101]=OFF时不可见）

事件页2（同行态）:
- 条件: S[101]=ON
- 图像: 谢临川行走图
```
