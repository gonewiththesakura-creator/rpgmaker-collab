## 2. 每张地图的功能分区

### M001 宗门广场

|---------|------|------|---------|

| **回溯路径** | 各边缘 | 避免卡死，自由探索 | 无事件，纯通行 |

## 4. 视觉/氛围技术方案（MZ可做）

### 色调方案

| 地图 | 主色调 | Tint Screen参数 | 情绪目标 |
|------|--------|----------------|---------|
| M001 | 暖色调 | (0,0,0,0) 或轻微暖色 | 安全、日常、归属 |
| M002 | 暖→冷过渡 | 入口(0,0,0,0) → 出口(-20,-20,0,0) | 从舒适到微妙不安 |
| M003 | 冷色调 | (-40,-30,-10,0) 暗蓝灰 | 压迫、神秘、危险 |

### 天气/光照/滤镜使用点

| 地图 | 效果 | 位置 | MZ命令 | 目的 |
|------|------|------|--------|------|
| M001 | 无特效 | - | - | 保持日常感 |
| M002 | 轻微Fog | 后段 | Weather Effect: Fog, 强度1 | 预示即将到来的危险 |
| M003入口 | Fog增强 | 峡谷 | Weather Effect: Fog, 强度3 | 视线受限，紧张感 |
| M003古碑 | 发光效果 | 古碑 | Show Animation / 自发光tile | 神秘感，视觉焦点 |
| M003战斗 | Tint变暗 | 战斗区 | Tint Screen(-60,-60,-60,0) | 压迫感，败退前兆 |

### 过渡节奏（安全→紧张→压迫）

```
M001（宗门广场）
├── 色调: 正常/暖
├── 天气: 无
├── BGM: 宗门日常音乐（平和）
└── 情绪: 安全、归属
    ↓ 进入M002
    
M002（宗门周边）
├── 色调: 渐变（入口正常→出口偏冷）
├── 天气: Fog强度1→2
├── BGM: 逐渐加入不安元素
└── 情绪: 微妙紧张、好奇
    ↓ 进入M003
    
M003（古碑区域）
├── 色调: 冷/暗
├── 天气: Fog强度3
├── BGM: 压迫感音乐
└── 情绪: 紧张、危险、神秘
    ↓ 败退后
    
返回M001（情绪释放）
```

### MZ原生实现命令示例

```javascript
// M002 渐变效果（并行处理事件）
// 事件页条件：玩家X坐标 > 一定值
Tint Screen: (R-10, G-10, B-5, 0)  // 逐步变冷

// M003 氛围设置（自动事件）
Tint Screen: (-40, -30, -10, 0)      // 冷色调
Weather Effect: Fog, 3               // 浓雾
Change BGM: 压迫感音乐

// 古碑发光效果（Show Animation）
Show Animation: 古碑位置, 发光动画ID

// 战斗前Tint（Troop Event）
Tint Screen: (-60, -60, -60, 0)      // 变暗
Wait: 30帧
Tint Screen: (-40, -30, -10, 0)      // 恢复（败退后）
```

## 5. 参考分析（热门RPG）

### 参考1：《星露谷物语》(Stardew Valley)

**可借鉴点1：清晰的功能分区**
- 村庄中心（NPC聚集）→ 农场（个人空间）→ 野外（探索/危险）
- 借鉴：M001作为"村庄中心"，M002作为"过渡野外"，M003作为"危险区域"
- 应用：M001聚集NPC，M002过渡氛围，M003集中危险事件

**可借鉴点2：视觉焦点引导**
- 村庄中心的社区中心、山顶的火车站都是视觉地标
- 借鉴：M001的宗门石碑、M003的古碑作为核心地标
- 应用：大地标吸引玩家，小调查点分散在周围

### 参考2：《去月球》(To The Moon)

**可借鉴点1：氛围叙事**
- 通过色调变化表达情绪（现实=彩色，记忆=暖色，痛苦=冷色）
- 借鉴：M001暖（安全）、M002过渡、M003冷（危险）
- 应用：Tint Screen渐变，不依赖复杂美术

**可借鉴点2：紧凑地图设计**
- 地图不大，但信息密度高，每个区域都有目的
- 借鉴：3张地图控制在20x20 tiles以内
- 应用：避免空旷，每个tile都有意义

### 参考3：《八方旅人》(Octopath Traveler)

**可借鉴点1：2D-3D混合的视觉层次**
- 前景、中景、背景分层，增加深度感
- 借鉴：MZ的Layer系统做前景遮挡、背景远景
- 应用：M003的峡谷做前景岩石，增加压迫感

**可借鉴点2：地标性视觉焦点**
- 每个区域都有独特的地标建筑
- 借鉴：M001宗门石碑、M003古碑作为记忆点
- 应用：独特tileset，发光效果，可调查

## 7. 风险与Fallback（5条）

| 风险ID | 风险描述 | Fallback方案 |
|--------|---------|-------------|
| R1 | **M003氛围做不出压迫感** | 简化方案：只用Tint Screen变暗，不用Weather；或直接用暗色tileset |
| R2 | **必败战斗触发不稳定** | Fallback：改为剧情事件（不进入战斗），直接播放败退动画 |
| R3 | **地图tileset素材缺失** | 使用MZ默认素材；宗门用"Outside"，古碑用"Dungeon" |
| R4 | **3张地图串联有bug** | 先做M001单图可玩，再逐步添加M002、M003 |
| R5 | **工时超支** | M002可简化（减少调查点）；M003氛围效果降级 |
