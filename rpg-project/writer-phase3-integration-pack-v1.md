## B. 文本压缩版 (MZ 可直接贴)

### M004 事件 1: 裂谷 QTE 后独白

```
你在做什么: 你拉着(同伴)的手，指甲抠进岩缝。
为什么: 因为你选择了救他，现在没有退路。
下一步: A. 全力向上(B 键 QTE)  B. 稍作休息(风险:二次塌方)

[后反思] (QTE 成功)
"值得...你weight的重量。"
变量: GS_RIFT_CLIMB_SUCCESS=true
```

**压缩原则**：
- 每屏 ≤3 行
- "你在做什么/为什么/下一步"三句式
- 反思句 ≤1 句，避免长独白
