# Writer Phase 3 Polish Lines v1

**Purpose**: Refine the core segments already integrated into the build (Top 8 + P0 events)
**Scope**: Only the texts that have been connected to the engine (M004-M008), no new content
**Focus**: 文句抛光、情绪力度校准、节奏修剪、收束句增强
**Constraint**: 不新增分支、不扩时长、不增复杂变量、只修已有段落

---

## 精修原则

1. **文句抛光**：让每句话更自然、更口语、更有角色语气
2. **情绪力度校准**：根据情绪目标（紧张/震撼/羁绊/紧迫）调整用词和节奏
3. **节奏修剪**：删冗余词，每屏严格 ≤3 行，加快阅读速度
4. **收束句增强**：每段最后一句必须有"钩子"或情绪定格

## 📖 1. 裂谷救援后三句独白 (M004 事件1) - P0

**原压缩版**:
```
你在做什么: 你单手抠着岩缝，另一只手还拉着同伴。
为什么: 因为你选了救他——现在没有退路。
下一步: A. 全力向上(B 键 QTE)  B. 稍作休息(风险:二次塌方)

[后反思] (QTE 成功)
"值得...你的重量。"
变量: GS_RIFT_CLIMB_SUCCESS=true
```

**精修版**:
```
你在做什么: 你指甲抠进岩缝，另一只手紧握同伴的手腕。
为什么: 你选了救他——现在要么一起上去，要么一起掉下去。
下一步: A. 全力向上(快速按键)  B. 稍作休息(二次塌方风险)

[QTE 成功后]
"值得...你的重量。"
变量: GS_RIFT_CLIMB_SUCCESS=true
```

**精修点**:
- **文句抛光**: "紧握" 比 "还拉着" 更有力；"一起上去，要么一起掉下去" 更口语化
- **情绪校准**: "要么一起上去，要么一起掉下去" 增强责任感和紧迫
- **节奏修剪**: 删"因为"前的停顿，更紧凑
- **收束句**: "值得...你的重量。" 保持简短有力

## 📖 2. 营地篝火三选一入口 (M005 事件1) - P0

**原压缩版**:
```
篝火噼啪。三个人: 老兵(抽烟)、药剂师(草药)、谢临川(擦剑)。

你在做什么: 决定先问谁——时间有限。
为什么: 每人知道一部分真相。
下一步:
  A. 问老兵("沈家军覆灭?")
  B. 问药剂师("青鸾未死?")
  C. 问谢临川("你的承诺?")
  D. 自己冥想(恢复 MP)

变量: GS_CAMP_FIRST_CHOICE=1/2/3/4
```

**精修版**:
```
篝火噼啪。三人: 老兵(抽烟)、药剂师(草药)、谢临川(擦剑)。

你在做什么: 决定先问谁——时间不多。
为什么: 每人知道一部分真相，但不会全说。
下一步:
  A. 问老兵("沈家军为什么覆灭")
  B. 问药剂师("青鸾未死是什么意思")
  C. 问谢临川("你的承诺是什么")
  D. 冥想(恢复 MP)

变量: GS_CAMP_FIRST_CHOICE=1/2/3/4
```

**精修点**:
- **文句抛光**: "时间不多" 比 "时间有限" 更紧迫；"不会全说" 增加悬念
- **情绪校准**: 暗示每个NPC都有保留，增加对话深度
- **节奏修剪**: 选项问题加"吗/什么"更自然；删"自己"直接"冥想"
- **收束句**: 选项列表清晰，让玩家快速决策
