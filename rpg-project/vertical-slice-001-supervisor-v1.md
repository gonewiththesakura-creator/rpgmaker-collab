# vertical-slice-001 监督定稿 v1

Date: 2026-03-18  
Project: 太玄界（RPGMaker MZ）
Supervisor: main node

---

## 1. 目标定义（本阶段唯一目标）

把第一阶段（前四章）做成一个**可玩、可测、可继续扩展**的 vertical slice：

- 一周目表层体验成立（宗门修行 + 试炼 + 线索 + 冲突）
- 第四章结尾明确出现“超出普通修仙RPG”的强钩子
- 但不提前透支完整多周目/残我系统

> 不是做完整系统，
> 是做“足够强的第一段体验 + 清晰的后续入口”。

---

## 2. 统一角色范围（已拍板）

前四章实装角色仅限：

1. 沈照璃（主角）
2. 谢临川（人间锚点）
3. 素问真人（关键引导）
4. 宗门师尊（功能型）
5. 表层反派：黑衣人
6. 绮罗音（残我遭遇态）

仅文本提及，不做系统实装：

- 涂瑶、寒见雪
- 绯铃、宁素心、白清漪、姬无瑕、原初照璃

---

## 3. 前四章统一流程（最终版）

### Chapter 1：宗门日常与入门异常
- 建立主角与宗门秩序
- 引入谢临川
- 体质异常第一次轻触发（不完全揭示）

### Chapter 2：试炼推进与关系加压
- 师尊发放试炼
- 谢临川剧情同行（方案B）
- 素问真人切入并加深“你不一样”的信息

### Chapter 3：古碑线索与表层冲突
- 接触沈家相关古碑线索
- 黑衣人正面对抗（章节高潮）
- 获得“残破家谱”物品线索（含“九劫归一”伏笔）

### Chapter 4：深渊入口与残我首次遭遇
- 情绪收束 + 压迫氛围
- 绮罗音遭遇（遭遇+对话+关键选择+结果标记）
- 结尾给出“残我密钥碎片”级别的强钩子与后续入口提示

## 4. 技术边界（最终版）

### 允许做
- MZ 原生事件为主
- 剧情同行（谢临川方案B）
- 绮罗音简化遭遇流程（不做完整Boss系统）
- 轻量关系开关
- 章节进度标记
- 物品描述承载线索

### 禁止提前做

- 完整遗钥系统
- 完整残我Boss体系
- 重插件依赖的UI/战斗改造
- 大规模分支对话树

## 5. 量化约束（执行控制）

- 事件总量目标：<= 400（建议控制在 330~380）
- 关键立绘角色：<= 5
- 最小变量/开关集：<= 30（前四章核心逻辑）
- 每章必须有 1 个情绪高点 + 1 个推进钩子

> 这些是控制线，不是炫技线。
